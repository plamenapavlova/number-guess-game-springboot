# Number Guess Game - Spring Boot

A simple number guessing game built with **Java** and **Spring Boot**.  
The player has to guess a number between **1 and 9** and only gets **3 tries**.  
The game also remembers how many attempts have been used so far.

## Features

- Guess a number from **1 to 9**
- Only **3 attempts** allowed
- Tracks the number of tries used
- Built with **Java Spring Boot**
- Runs on **localhost**

## Technologies
Java • Spring Boot • MVC • Thymeleaf • Maven

## How It Works

1. The game generates or stores a number between **1 and 9**
2. The player submits a guess
3. The application checks whether the guess is correct
4. The player can try up to **3 times**
5. The game keeps track of how many attempts have been made
